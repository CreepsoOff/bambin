# Bambin 👨‍👩‍👧‍👦

Bambin est une application iOS développée en **SwiftUI**.

Elle permet aux familles de découvrir des activités à faire avec les enfants : cuisine, sport, nature, dessin, sciences, et bien plus encore.

## Présentation

L’application propose une interface simple et colorée pour trouver rapidement une activité adaptée à toute la famille.

Les activités peuvent être consultées, filtrées ou recherchées selon les envies. Chaque activité possède une fiche détaillée avec les informations utiles : durée, difficulté, matériel nécessaire et nombre de participants.

## Fonctionnalités ✨

* Découverte d’activités familiales
* Recherche d’activités
* Filtres par catégories
* Fiche détaillée pour chaque activité
* Gestion des membres de la famille
* Ajout d’un membre avec avatar et centres d’intérêt
* Suivi des favoris et des activités réalisées

## Technologies 🛠️

* Swift
* SwiftUI
* PhotosUI
* Xcode

## Installation

Cloner le projet :

```bash
git clone https://github.com/CreepsoOff/bambin.git
```

Ouvrir le projet dans Xcode :

```bash
cd bambin
open bambin.xcodeproj
```

Lancer l’application avec :

```bash
Cmd + R
```
